# 시작하기
1. python을 설치하세요.
2. 설치하는 창에서 왼쪽 밑 add to .PATH를 클릭하여 설치합니다.
3. 폴더를 하나 만드세요.
4. 거기에 소스코드를 다운해서 넣으세요.
5. 소스코드를 열고 맨 밑으로 내려 토큰을 추가하세요.
6. 저장을 한 후 파일 탐색기 주소에 cmd라고 치세요
7. python verified.py 를 입력하세요
8. 그다음 세팅 하시면 됩니다.

# 세팅하기
1. /세팅하기 명령어 치시면 세팅 가능합니다.
